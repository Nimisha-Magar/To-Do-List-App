# To-Do-List-App

A clean, modern task management application built with HTML, CSS, and JavaScript. Features include:

Add Tasks - Quickly add new tasks using the input field or press Enter
Track Progress - Visual progress bar and counter showing completed vs. total tasks
Mark Complete - Check off tasks with an animated checkbox
Edit Tasks - Modify task text (disabled when marked complete)
Delete Tasks - Remove tasks with a single click
Persistent Storage - Tasks are saved to browser localStorage and load on refresh
Responsive Design - Works seamlessly on desktop and mobile devices
Empty State - Shows an image when no tasks exist
Confetti Celebration - Animated confetti effect when you complete tasks
Tech Stack: HTML5, CSS3 (Flexbox), Vanilla JavaScript, Font Awesome Icons, Local Storage API

Perfect for daily task management and productivity tracking!
